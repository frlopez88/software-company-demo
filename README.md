# software-company-demo
This is a project for a fictional software company designed to show students the process of building a project and collaboration with fellow developers. 

## How it's Made
Frontend: 
- HTML
- CSS
- Bootstrap

Backend:
- JavaScript
- Supabase for storage

## Design Tools Used

- Icons: [Iconify](https://iconify.design/)
- Color Palette: [Coolors](https://coolors.co/)
